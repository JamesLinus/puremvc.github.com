class org.puremvc.haxe.multicore.utilities.pipes.plumbing.TeeSplit
implements org.puremvc.haxe.multicore.utilities.pipes.interfaces.IPipeFitting

Splitting Pipe Tee.

Writes input messages to multiple output pipe fittings.

function new(?output1 : org.puremvc.haxe.multicore.utilities.pipes.interfaces.IPipeFitting, ?output2 : org.puremvc.haxe.multicore.utilities.pipes.interfaces.IPipeFitting) : Void

Constructor.

Create the TeeSplit and connect the up two optional outputs. This is the most common configuration, though you can connect as many outputs as necessary by calling connect.

function connect(output : org.puremvc.haxe.multicore.utilities.pipes.interfaces.IPipeFitting) : Bool

Connect the output IPipeFitting.

NOTE: You can connect as many outputs as you want by calling this method repeatedly.

function disconnect() : org.puremvc.haxe.multicore.utilities.pipes.interfaces.IPipeFitting

Disconnect the most recently connected output fitting. (LIFO)

To disconnect all outputs, you must call this method repeatedly untill it returns null.

function write(message : org.puremvc.haxe.multicore.utilities.pipes.interfaces.IPipeMessage) : Bool

Write the message to all connected outputs.

Returns false if any output returns false, but all outputs are written to regardless.
