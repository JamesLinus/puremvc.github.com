Classpublic class RetryParameters
ImplementsIRetryParameters

See this utility's RetryPolicy class as the consumer of RetryParameters. See the demo app called LoadupAsOrdered as an example of how RetryParameters can be used.

RetryParameters is a way of encapsulating the configuration parameters required by the RetryPolicy class. A RetryParameters object is immutable.
