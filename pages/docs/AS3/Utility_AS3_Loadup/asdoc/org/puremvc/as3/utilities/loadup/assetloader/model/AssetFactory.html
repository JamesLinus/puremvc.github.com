Classpublic class AssetFactory
ImplementsIAssetFactory

This factory produces IAsset objects via its getAsset method. The inputs are the asset url and an optional type. When a type is not given, it is calculated from the url, based on the url string ending, as follows
  • .jpg, .gif, .png => Image
  • .txt, .xml, .css => Text
  • .swf => Swf
  • .flv, .f4v => Video
  • .mp3 => Audio
  • any other alphabetic ending, e.g. .abcde, => Text
  • otherwise, e.g. no ending, => Error is thrown.

At instantiation, the factory is given an IAssetTypeMap instance. Given the asset type, this map is used to lookup the Class to use to instantiate the IAsset object.
