remarks
In PureMVC, IProxy implementors assume these responsibilities:Implement a common method which returns the name of the ProxyAdditionally, IProxys typically:Maintain references to one or more pieces of model dataProvide methods for manipulating that dataGenerate INotifications when their model data changesExpose their name as a public const called NAMEEncapsulate interaction with local or remote services used to fetch and persist model data
